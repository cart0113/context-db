This project has a context-db knowledge base at `context-db/`.

Use `/context-db <command>` to interact with it:

- `/context-db prompt "<instruction>"` — consult for relevant
  knowledge/standards
- `/context-db pre-review "<plan>"` — check plan against standards before
  implementing
- `/context-db review "<summary>"` — review changes against conventions
- `/context-db update "<learnings>"` — file learnings into context-db
- `/context-db maintain` — audit and maintain context-db quality
