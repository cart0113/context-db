When starting a session or when you need to understand this project:

1. Read `CONTEXT/CONTEXT_MD_SYSTEM_INSTRUCTIONS.md`
   - This file explains the context-md system and how to use it
   - It contains project-specific guidance for this codebase
   - It tells you where to find additional context

2. Read `CONTEXT/CONTEXT_TOC.md`
   - This is the auto-generated index of all available context
   - It lists subfolders and files with one-line descriptions

3. Fetch only what you need
   - Do not load all context upfront
   - Progressive discovery keeps your context window focused
